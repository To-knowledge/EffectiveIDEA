# EffectiveIDEA
Idea hotkeys for Win.

## IDEA 鼠标脱离计划
### Win版常用快捷键

### 跳转

`ctrl + E` 查看最近修改的文件集合

`alt + 1` 左侧文件夹与右侧代码区切换

`ctrl + N` 查找相关类



### 文件内上下文

`shift + enter` 新开一行

`ctrl + Y` 删除当前行

`ctrl + shift + up/down` 上下移动当前行

### 快速编码

`alt + insert` 插入构造方法，getter以及setter等

`alt + enter`  代码修复建议等、自动构造函数、下一步提示等
